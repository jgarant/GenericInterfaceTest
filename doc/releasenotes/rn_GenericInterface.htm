GenericInterface v1.0

Esta es la versión inicial del componente.

Autor: Fernando Castaneda / Javier Flores / Mauricio Rodriguez
Fecha: 2019.12.10
Entidad: Banco de Bogotá
Instalador: GenericInterface_v1.0_build1912101_r5.6.exe
Descripción Funcional: Interfaz generica source que se encarga de administrar los mensajes Base24 e Iso8583 que se trasmiten entre la la red bancaria y Postilion.
Descripción Técnica: Es una interfaz que posee los métodos correspondientes para el procesamiento de los mensajes que viajan desde la red externa hacia postilion. Esta interfaz recibe el mensaje y lo transforma a Base24, posteriormente realiza la conversión a formato Iso8583 teniendo en cuenta una serie de validaciones por campo con expresiones regulares y basandose en la configuración del mensaje que se encuentra parametrizado en la base de datos. Si el mensaje contiene algun error, la interfaz se encarga de notificar a TM por medio de un 0220 el error detectado, detallado en el campo 127.22 y se declina la transacción a la red externa con un 0210 con error de formato (P39 = 30). Si la transacción pasa las validaciones se envia el mensaje hacia postilion, cuando postilion responde el 0210 ISO, la interfaz se encarga con el mismo proceso de armar el 0210 B24 para ser enviado a la red externa. Esta interfaz maneja el mensaje 0220 Notificacion de Bloqueos, donde la entidad remota nos notifica que la tarjeta ha sido bloqueada, y se notifica dicho mensaje para que en el proceso del Extract salga este reporte y el Banco pueda cobrar una comisión al tarjeta habiente.
