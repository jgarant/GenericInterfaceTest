Support Team

Tipicamente <none>

    Parámetros de Usuario:

    Parametro

    Descripción

    Tiempo de Retención

    Tiempo en milisegundos que la interfaz va a mantener en memoria los mensajes que han llegado.

    Validación de MAC

    Valor lógico. Si es true indica validación de MAC de lo contrario no realiza la validación.

    LLave KWA

    Llave para realizar la validación de MAC.

    Envio SignOn hacia Remoto

    0/1: Se coloca el valor de 0 para que cuando la interfaz inicie y se conecte a remoto no genere ni envie el mensaje 0800 de tipo SIGN ON. Se coloca el valor de 1 para que cuando la interfaz inicie y se conecte a remoto genere y envie el mensaje 0800 de tipo SIGN ON.

    Validaciones de negocio

    true, si desea realizar validaciones de negocio, de lo contrario false.

    Envio notificaciones

    true, si desea enviar notificaciones hacia Transaction Manager de las declinaciones realizadas por la interfaz, de lo contrario false.

    Issuer

    Número del issuer dentro de Postcard. Para la consulta de información de cuentas.

    IP logger

    IP del servidor UDP para guarda la información de monitoreo de la interfaz.

    Puerto logger

    Puerto del servidor UDP para guarda la información de monitoreo de la interfaz.

    Consulta Convenios

    true, para realizar consulta de convenios en la interfaz.

    Codificar Info

    true, para codificar la informaci√≥n sensible de los tags del campo 127.22 en base 64.

    Enrutamiento

    0, para enrutar en base al modulo del campo 11, si es impar enruta hacia la interchange con el puerto de ahorros del coordinador, si es par enruta hacia la interchange con el puerto de corriente del coordinador. 10 para enrutar hacia la interchange con el puerto de ahorros del coordinador. 20 para enrutar hacia la interchange con el puerto de corriente del coordinador.

    URL CUT

    http://ateneams07d.banbta.net:31301/api/entities?name=TxHomologadaxAplicacion, URL para establecer conexión a servico REST y realizar consulta del CUT.

    IP Token

    IP que consumimos para el servicio de activación de token.

    PORT Token

    Puerto que consumimos para el servicio de activación de token.

SAP

En el panel SAPs, creamos una SAP nueva, el siguiente cuadro muestra las principales configuraciones a realizar en esta nueva SAP:

Variable

Setting

SAP

Es el nombre que va a tener la SAP. El producto genera uno por defecto. Se sugiere dejar el mismo.

Type

Seleccione server. Pues la interfaz va esperar conexiones desde la NixBridge Banco de Bogotá.

Protocol

Seleccione (TCP/IP) Postilion.

IP Address

No se especifica ninguna. Pues como se definio como tipo server, la interfaz solo va a esperar conexiones.

Port Number

Cualquier puerto valido. Verificar que no este uso.

Max Connections

Especificar el máximo de conexiones que va permitir.
